# Ichin Browser

A modern desktop web browser interface inspired by next-generation productivity browsers.

## Features

- Clean, premium, minimalist dark theme aesthetic
- Vertical sidebar with user profile, workspaces, and tabs
- Workspace switcher with colored icons
- Pinned and active tabs sections
- Favorites/bookmarks section
- Command palette (⌘K)
- Split-view support for multiple webpages
- Collapsible sidebar
- Floating tab previews on hover
- Minimal toolbar with navigation controls

## Components

- **Sidebar**: Main navigation with tabs, workspaces, and user profile
- **Toolbar**: Navigation controls and URL bar
- **WebView**: Displays webpage content
- **CommandPalette**: Quick command access overlay
- **TabItem**: Individual tab with hover preview
- **WorkspaceSwitcher**: Colored workspace icons

## Maintenance Notes

When updating this project, ensure to:
1. Update this README if new features are added
2. Maintain consistent dark theme variables
3. Keep components modular and under 80 lines each
