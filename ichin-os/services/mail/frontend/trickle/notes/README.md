# Ichin Mail

A modern, minimal, lightweight email client frontend for the Ichin Network.

## Features

- Clean, distraction-free interface
- Three-panel layout (Sidebar, Email List, Email View)
- Dark and Light mode support
- Compose new emails
- Search and filter emails
- Collapsible sidebar
- Settings page

## Pages

- `index.html` - Main email client interface
- `settings.html` - User settings and preferences

## Design Philosophy

- Inspired by Apple Mail, Proton Mail, and Linear
- Neutral color palette with blue accent
- Focus on readability and productivity
- No unnecessary animations
